Hello! This is our data analysis project!

Тема проекта: анализ data-related вакансий (аналитики, инженеры), а также вакансии программистов в России. 

ML: 
1) предсказание сферы, в которой сможет работать человек с заданными навыками
2) прогноза зп по всем признакам (локация, навыки, формат работы и тд)

Что мы сделали на данный момент:
1) спарсили через API ссылки на вакансии на hh ru.
2) Проитерировались по ссылкам и спарсили данные с каждой страницы
3) Обработали данные и почистили их от выбросов и пустот
4) Сделали EDA
5) Выдвинули гипотезы
